# homelab

## Overview

This repository contains my personal enterprise homelab project built to improve my Linux System Administration and DevOps skills.

The goal is to simulate a small enterprise environment using virtual machines and open-source technologies.

## Technologies

- Ubuntu Server
- Windows Server
- Docker
- Docker Compose
- Grafana
- Prometheus
- Active Directory
- DNS
- DHCP
- VMware
- Git

## Project Goals

- Deploy Linux and Windows infrastructure
- Configure Active Directory
- Build Docker environment
- Monitor servers
- Automate administration tasks
- Document the entire infrastructure
